Homework 2: Preparation

This page contains some hints on how to use LIONsolver and Python for solving the exercises of homework 2 ( Learning from Data course ).

Hoeffding Inequality

The first exercise is about implementing a computer simulation for flipping a set of coins. Here you can find a Python template which you can interface with LIONsolver as a Parametric Table, (as you did for the previous homework). The script requires three input parameters:

  • Number of tests (default 100000)
  • Number of coins (defualt 1000)
  • Number of times to flip each coin (default 10)

The script outputs a table containing the frequency of heads (fictional, for now) obtained by the three coins specified by the exercise (c_1, c_Rand, c_Min).

The average can be obtained by plotting the content of the table with a bubblechart and enabling the Polynomial fit of degree 0.
